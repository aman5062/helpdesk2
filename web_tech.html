Web Technology

Web Server : Web communicatation takes place between a client process and the web server. The server process creates a socket and client process access the server through the socket mechanism.
  1. HTTP server process is created on a port, which wait for clients to establish TCP (Transmission Control Protocol) connection.
  2. An HTTP client initiate a TCP connection with HTTP server at the deginated port.
  3. The HTTP server accept this connection.
  4. The HTTP client send a request for a resources to the server.
  5. The HTTP server closes the TCP connection.
  6. The HTTP client receive the response containing information and process it.

File organisation
  1. Ordinary file :- An ordinary file consists of characters stored in the disk.
  2. Special file :- A special file is always representing devices such as terminal, printer etc.
  3. Directory:- It is a collection of files and other sub directories.
  4. Bin :- It contains executable files for booting the process and system software.
  5. Dev :- It contains the special files that represent devices like terminal, console, printer etc.

OS commands
  1. ls :- Displays information about files in the current directory.
  2. mkdir :- Creates a directory.
  3. rmdir :- Removes empty directories from the directory lists.
  4. chmod :- allows an administrator to set or modify a file permission.
  5. cd :- To navigate between different folders.
HTML commands <html>: beginning and end of an HTML document. <head>: header information of the document. <title>: specifies the title of the page. <body>: contains the main content. <h1> to <h6>: define headings of different sizes. <p>: This tag defines a paragraph of text. <a>: creates a hyperlink. <img>: inserts an image into the page. <ul>: create an unordered list. <ol>: create an ordered list. <br>: inserts a single line break <hr>: the element is displayed as a horizontal rule that is used to separate content.
